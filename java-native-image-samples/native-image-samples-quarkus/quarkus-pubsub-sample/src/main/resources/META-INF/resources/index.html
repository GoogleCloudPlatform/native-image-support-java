Pub/Sub Quarkus Application

This is a Quarkus application which demonstrates basic operations with Cloud Pub/Sub. You can also manage the Pub/Sub resources created here through Google Cloud Console.

Pub/Sub Admin -- create

Create topic
Create subscription for topic

Pub/Sub -- post and subscribe

Send or retrieve/ack one message at a time
Message Retrieval (acks one message at a time)
Send copies of message to topic

Pub/Sub Admin -- delete

Delete topic
Delete subscription
